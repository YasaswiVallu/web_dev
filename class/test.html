HTML

HTML stands for HyperText Markup Language. It's the foundation of the web, used to structure and organize content on web pages by using tags and attributes.

CSS

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML.

JAVAScript

JavaScript is a programming language primarily used to make web pages interactive. It's often used alongside HTML and CSS .

Web designing

Web designing is the process of planning and arranging content on a website to create a visually appealing and user-friendly experience.
